Provides various utility applications and classes, in addition to what is included in the utilities package.

This package currently provides the following utilities:

  • CommandHelpRetriever - Allows you to generate HTML documentation for a given CL command (*CMD) name, library, and server. This function is available both as a Java program and as an application programming interface (API).
