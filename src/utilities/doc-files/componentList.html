Components supported by IBM Toolbox for Java

Following are the components IDs that you may specify when invoking the ToolboxJarMaker tool.

  • The first column lists the common name for the component.
  • The second column lists the keyword that you should specify when using the -component option tag.
  • The third column lists the Integer value that you should specify in setComponents() and getComponents().
